# 1.5 Test Deliverables

The following documents, reports, and artifacts will be delivered during and after the testing process:

## Documents
- Test Plan Document  
- Test Scenarios & Test Cases (Excel/Markdown format)  
- Test Data Sheets  
- Bug/Defect Reports  

## Reports
- Daily/Weekly Test Execution Reports  
- Defect Summary Report  
- Test Coverage Report  
- Final Test Summary Report  

## Artifacts
- Screenshots & Logs (for failed test cases)  
- API Test Collections (Postman)  
- Performance Test Scripts (JMeter)  
- Security Test Reports (Burp Suite findings)  

## Version Control
- All deliverables will be stored in the GitHub repository under their respective sections.  
- Each deliverable will be versioned and updated with change history.
